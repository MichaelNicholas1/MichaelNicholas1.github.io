Champion Mastery Comparison

This personal project (which is currently on hold) started in my spare time back in Winter of 2019. I ended up contributing more towards the project starting in December of 2020, where I started to take it more seriously to prepare myself for future roles in software engineering.

The project is a proof-of-concept application hosted on Heroku that allows for players of the popular MOBA game League of Legends to compare their character data and mastery with other players, to see who is the best at what characters. It uses Riot Games' developer API to collect player data and champion statistics, stores them into a MySQL relational database, and displays them in a PHP frontend application.

In it's current state, the program is quite barebones. It displays the character data of a single user at a time, but I really need to kick this project back into gear and start to include new characters and features that have released since I last worked on it! (Please keep in mind that that the API key for this website has to be regenerated manually each day, so please contact me if you would like me to update the key so that you can see the product in action).
